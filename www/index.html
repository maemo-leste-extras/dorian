Dorian E-Book Reader

Dorian is an e-book reader for EPUB format books. It supports book reading, organizing books into library, bookmarks, color and font styles, portait and landscape modes.

The application can be downloaded using the N900's Application Manager, after activating the Maemo extras-testing repository. Development (read: unstable, exploratory, but with lots of new features) versions are available from the extras-devel repository.

The project is maintained in the Maemo Garage.
